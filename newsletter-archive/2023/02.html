Developer Resources
Adyen Adobe Commerce Plugin
We have renamed the Adyen Magento 2 Plugin to Adyen Adobe Commerce Plugin. This change was made in light of our new partnership and to better reflect the compatibility of the plugin with all three platform versions, including Magento OS, Adobe Commerce, and Adobe Commerce Cloud.

We believe that the updated name is more relevant to merchants who use Adobe Commerce plugins and will help boost its popularity.

Validating API requests and responses
Validating your API specification is a must, but it is also crucial to ensure the API "does what it says" and honours the API contract.
Read this blog about runtime validation of API requests and responses using OpenAPI and Postman, a developer tool for validating REST APIs and a few best practices to shift left.

Consuming webhooks
We have published a new article to help you setup, validate and consume webhooks on the Adyen platform.

This is a developer guide for subscribing and processing asynchronous events that deliver essential information like payment updates, configuration changes, reporting and many others. The blog shows how to define, secure and implement a webhook handler: look at the code snippets, run a working sample and learn the best practises for an optimal integration.
