New Authorisation Adjustment Integration-Example
We've added the Authorisation Adjustment use case to our integration-examples on GitHub. The example features a hotel booking use-case and includes five main operations:
  1. Pre-authorisation /update-payment-amount API Call
  2. Authorisation adjustments: increase or decrease an amount after the pre-authorisation
  3. Extend expiry dates of the authorisation
  4. Capture the pre-authorisation
  5. Reversals: cancel or refund the payment
Visit integration-example on GitHub ›

Libraries & Plugins
If you've missed the announcement, we're now using the Adyen OpenAPI specifications to generate all models and services for our libraries. This guarantees that our libraries are consistent and in-sync with the latest version (v70) of the Adyen API endpoints.

In the following section, we'll highlight some of the important release notes. For the major releases, we've included a migration guide for our developers.

Library/Plugin Highlights
Adyen Web 5.49.4
Added missing ResultCodes in PaymentMethodData: AuthenticationFinished, AuthenticationNotRequired, PartiallyAuthorised, PresentToShopper, Received, RedirectShopper and Refused.
Adyen Java 21.1.0
Switched back to using Jackson as the serialization framework for the Java library, reverted the changes from GSON.
Models/services are updated to match the latest version (v70) of the Adyen API. A migration guide can be found in our release notes on GitHub.
Adyen .NET 11.0.0
Added support for grant payouts from the Adyen Capital API.
Models/services are updated to match the latest version (v70) of the Adyen API. A migration guide can be found in our release notes on GitHub.
Adyen Node.js 14.0.0
Models/services are updated to match the latest version (v70) of the Adyen API.
We've included a migration guide in our release notes.
Adyen PHP 15.0.0
Models/services are updated to match the latest version (v70) of the Adyen API.
We've included a migration guide in our release notes.
Adyen Go 7.1.0
Added support for Balance Platform and Capital API endpoints.
Adyen Python 9.0.1
Added function that fetches the terms of service status from the Adyen Legal Entity API.
Adyen iOS 5.3.0
Apple Pay: ApplePayConfiguration now supports the capabilities of PKPaymentRequest. PKPaymentRequest represents a request for payment, including details about payment-processing capabilities, the payment amount, and shipping information.
Adyen Magento2 8.20.0
Apple Pay: Added support for tokenization.
