New Adyen Test Card Chrome Extension
Testing payment flows can be a time-consuming task, as it involves using different payment methods and typing/pasting card numbers or other details. This is why we built a Chrome extension tailored to simplify the testing process and remove the burden of continuously visiting the Adyen Test Cards page, or keeping a long list of card numbers on a local file. Get it now from the chrome web store or install directly from GitHub.

Tools and Resources
Lume is now Open Source
Our frontend team recently open-sourced an internally built project called Lume as part our commitment to give back to the Open Source community. Lume is a combination of design guidelines & patterns for easy data visualization, and a Vue component library that implements them. Check it out, spread the word, use it, contribute and help improve the project.

Node.js and .NET Giving sample
The Adyen Giving integration-example which gives shoppers the option to donate to a charity as part of a payment flow is now available in Node.js and .NET. Check it out on our adyen-example Github page.

Dispute API collection now available
We've added a new Postman API collection for Disputes. You can use the Disputes API to automate the dispute handling process so that you can respond to disputes and chargebacks as soon as they are initiated.

Libraries and Plugins
Checkout Android Library v5.0.0 release
We are happy to announce the v5.0.0 release of the Checkout Android library. This version includes a simplified integration for Drop-in and Components using the /sessions endpoint, brings support for 9 new payment methods and significantly reduces the integration effort with Components. Note: During our previous livestream with GooglePay Devs, this release was in Beta but is now officially available as a major release. We invite developers to check out the new release and share feedback to help us improve it!

New Cash App payment method for SFRA merchants
Popular North America payment method Cash App Pay is now available in the latest release of Salesforce Commerce Cloud cartridge for SFRA merchants. This allows merchants to reach a rapidly growing customer base in the USA. Check out the release note for more.
