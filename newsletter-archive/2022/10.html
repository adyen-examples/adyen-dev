Hi folks,

Here is the October 2022 Developer's Newsletter: blogs, conferences, events and new releases of our libraries and plugins. Guess what? I am a new developer advocate and here is a quick introduction:

Hi! I am Ayodeji Ogundare, Developer Advocate here at Adyen and I will be focusing on the E-commerce Integrations. I am a PHP Developer and Community Builder. Over the years, I have built and lead developer communities for Google, WordPress, Codecademy and Polygon. I love and advocate open source! I look forward to working with you to help make integrating with Adyen a lot easier. You can reach out to me on Twitter and LinkedIn.

Latest Releases
Adyen-Web  5.28.1 
  • Locked the Preact version used by the SDK, forcing the client to use the same version as we use for development
  • Added Card config property to force the use of the JWE compat version
  • Upped SecuredFields version to 4.3.1
  • Added Payment method: PayByBank
  • Added support for `enableStoreDetails`
  • Payment submission errors for Google pay now trigger the `onError` callback instead of throwing an exception
  • Added Online Banking PL (P24) component
  • UPI redesign
Adyen-PHP-Api-Library  13.0.4 
  • Added deprecation tag to `setLogger` and `getLogger` functions
  • Fixed for region validation & added India to the regions list
  • Added checkout technical cancels
Adyen-Java-Api-Library  18.1.1 
  • Allowed client configuration via system properties
  • Added `AuthenticationNotRequired` payment result code
  • Added OpenAPI model templates
Adyen-Dotnet-Api-Library  9.1.0 
  • Set TLS12 for both sync and async request
  • Added brand field to DefaultPaymentMethodDetails
  • Added obsolete warning to inform about HPP deprecation
  • Added async support for the Account Service
  • Added support for donations endpoint
Adyen-Go-Api-Library  6.0.1 
  • Upgraded Checkout API version to v69
Adyen-Python-Api-Library  7.1.1 
  • Fixed incorrect detection of paymentsRefunds action
Adyen-Android  4.8.0 
  • Added online Banking Poland Component
  • Updated 3DS2 SDK version to 2.2.8
  • When coming back from a redirect after drop-in was closed, we gracefully terminate drop-in
Adyen-Magento2  8.7.0 
  • Implemented combo card funding source selection for POS payments
  • Allow manual configuration of the HMAC key
  • Fixed External Platform Integrator to Adyen
  • Disabled cross button and ESC key on action modal
  • Fixed comparison when handling capture webhook from CA
  • Changed the way we are comparing payment amounts in handleCaptureWebhook() 
  • Fixed auto configuration fails on local test environment
  • Do not attempt to get brand for PayByLink payments
  • Handled Capture Requests triggered from Customer Area
Adyen-Shopware6  3.7.0 
  • Added new payment methods: Multibanco, MB Way, WeChat Pay (QR + Web), Affirm, PayBright, Vipps, MobilePay
  • Fixed origin issue with multiple domain in one storefront
