HMAC validation across all API libraries

All our API libraries now support HMAC validation. But why is HMAC so important? HMAC (Hash-based message authentication code) provides security by ensuring data integrity and authenticity between parties. By integrating HMAC, you're protecting your server from unauthorized webhooks events. Check out our documentation to learn how to implement HMAC in your integration.

Webhook configuration for Relayed Authorization now available in Balance Platform

Exciting update for Adyen customers with Issuing integrations: You can now configure webhooks for relayed authorization directly within the Balance Platform Customer Area. This feature allows for more streamlined and responsive transaction processing, ensuring you have real-time updates and control.

Hyvä Adyen Compatibility Module for Adobe Commerce is now live

We're excited to announce the release of the Hyvä Compatibility Module for the Adobe Commerce Plugin! This new module enhances your Adobe Commerce (Magento 2) ecommerce platform. It integrates with Hyvä Checkout, delivering a streamlined, efficient, and visually appealing checkout process.

Key Features:
Supported Payment Methods - credit/debit cards, digital wallets (Apple Pay, Paypal), Klarna and tokenization.
Payment Options - supports installments for card payments.
Adyen Giving - allows customers to make donations during their purchase process.

Check out our documentation for using Hyvä Checkout through the Compatibility module.
