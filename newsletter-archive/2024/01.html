More Adyen APIs are now available in API logs
Debug your Adyen for Platforms API calls without needing to instrument your code extensively. Log in to your Adyen Customer Area and go to the Developers section to check the API logs.

In addition to logs for your Checkout and Management API calls, logs for the following APIs are now available: Legal Entity Management, Configuration, and Transfers API. The logs show details you need for troubleshooting, such as the request ID, the resource ID, timestamps, the API credential used for the call, and the actual request and response.

Should you need further help from others or our Support team, you can copy the log link and send it to them to share the details. Are you having problems accessing the logs? Reach out to your Adyen admin user to get the required roles.

Libraries and Plugins
Adobe Commerce payment plugin v9
Explore the latest release of the Adyen Payments plugin for Adobe Commerce (formerly Magento), now with a simplified structure, including the renaming of "Required settings" to "Initial setup" and the consolidation of payment configurations under "Accepting payments." This upgrade introduces support for installments in tokenized payment methods, multi-shipping functionality for Klarna payments, Adobe Commerce Vault integration for token references, a new native gift card flow, and improved Adyen integration. Follow the steps in the migration guide to plan your upgrade now and get the plugin from our Github repo.
