Discover the Adyen API workflows with Postman Flows
Do you want to learn Adyen API workflows? This is possible using Postman Flows, a visual API workflow editor. It enables the creation of custom workflows by chaining API requests.
We provide and maintain several flows in the Adyen Dev Postman space. You can easily set up the environment, choose the desired flow, and run it with your API key.
With Flows, you can:
  • Learn through the visual representation of all steps involved in a workflow that requires multiple API calls.
  • Zoom in to inspect the underlying Postman request for each step.
  • Run the workflow using your API key for testing purposes.
  • Extend or modify existing requests to tailor them to your needs.

Visit the Adyen Dev Postman workspace to learn more and explore our Checkout with donation, Adyen for Platforms and In-Person payments flows.
