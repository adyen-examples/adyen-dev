Updates
Automatic deletion of old disabled webhook configurations
To help speed up the delivery of webhooks and reduce noise during monitoring, webhook configurations that have been disabled for more than six months will now be permanently deleted. We'll give merchants with a 30-day notice before automatically deleting the old disabled configuration. The warning message will also be displayed on the webhook list as well as, the view and edit pages. Read more 

Salesforce Commerce Cloud - B2C Commerce v24.1.0
SFCC v24.1.0 brings a streamlined cartridge repository, upgraded Web Components, and Checkout API for better performance. The business manager now reminds you to align Live URL prefixes with regions, with increased test coverage to ensure reliability. Level 2/3 scheme data is now exclusively sent for card payments. You can now use multiple live URL prefixes in a single sandbox, and US shoppers can use Venmo, which has an option to hide it. Additionally, Adyen Giving is available for Apple Pay donations, and the card payment method group is now labeled "Cards" in compliance with EU regulations. Check out more details in the release note.
